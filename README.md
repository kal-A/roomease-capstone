# RoomEase

A room booking and management web app built as a University of Waterloo engineering capstone project. RoomEase streamlines the process of finding, requesting, and approving campus spaces — replacing a fragmented manual process with a structured, role-based system.

**Tech stack:** Next.js 16 · React 19 · TypeScript · Supabase · Tailwind CSS · NextAuth

**Core features:**
- Room search and filtering based on capacity, AV equipment, and booking constraints
- Multi-step booking flow with ranked room recommendations
- Admin portal for request review, approval, and override handling
- Role-based access for users and administrators
- Analytics dashboard with booking and usage metrics

**Validation results from controlled user testing:**
- Task completion (usability): 50% → 88%
- User satisfaction: 33% → 90%
